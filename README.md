# IgExport
An application that exports Instagram Direct Messages. Easy and simple to use.

### Features
- Export entire chats
- Search entire chats
- Download all media from chats
- Print entire chat logs

### Supported message types
- Text messages
- Likes
- Post shares
- Story shares
- Images
- Videos
- Disappearing images and videos
- Voice messages
- GIFs
